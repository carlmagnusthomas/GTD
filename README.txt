Ground Truth Dataset (GTD)

This repository contains the files binary labelled datasets used to identify European summer blocking events. It can be extended for other regions and seasons. The labelling was done from studying geopotential height maps across five days within JJA Europe. More detail can be found in Thomas et al. (2021). "An unsupervised learning approach to identifying blocking events: the case of European summer". WCD in review. https://doi.org/10.5194/wcd-2021-1
